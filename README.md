文件
    文件保存在持久化存储设备中的一段数据。
    从功能的角度分为文本(打开后会自动解码为字符串)和二进制文件（视频，音频等）
    Linux下一切皆文件
    python中一切皆对象
    Python中把文件也视为一种类型的对象


字节串(bytes)
    字节串与ｓｔｒ不同，字节串以字节序列值表达数据，更方便的来处理进程数据。
    普通的ascii

    'alksdhnqwpoeulkdasf@#$%^&*())234567890-'   #字符串
    b'alksdhnqwpoeulkdasf@#$%^&*())234567890-'  #字节串
    字符串转换为字节串的方法：str.encode()
    字节串转换为字符串的方法：bytes.decode()



使用Python代码实现文件读写操作
文件读写操作步骤分为三步
１、打开文件
    open(file_name, access_mode='r')
    功能：打开一个文件，返回一个对象
    参数：
        file_name 　     文件名
        access_mode     打开文件的方式，如果不写默认为r
        r   以读方式打开文件，前提是文件必须存在
        w   以写方式打开文件，如果文件不存在就创建文件
        a　　　以追加方式打开文件
        r+  以读写方式打开文件，文件必须存在

    返回值：返回一个文件对象
２、读取文件
    read(size)
    功能：
        直接读取文件中的字符
    参数：默认为负数
    返回值：
        返回读取的内容

    注意：不建议一次性读完大文件

    readline()
    功能：读取一行
    参数　　如果不给定，则默认为读取一行
    返回值：返回读取到的内容

    readlines()
    功能：读取每一行作为列表的一项
    参数：如果不给定，则默认为读取一行
    返回值：返回读取到的内容
